parentNode
childNodes[nodenumber]

Here’s a clear flowchart representation:

Plaintext          ┌───────────────────────┐
          │       Current Node     │
          └───────────┬───────────┘
                      │
        ┌─────────────┼─────────────┐
        │             │             │
        ▼             ▼             ▼
  parentNode   firstChild     lastChild
     ▲             │             │
     │             ▼             ▼
     │     childNodes[0]   childNodes[length-1]
     │
     │
     ┌───────────────────────────────────────┐
     │ From any node, you can also navigate:  │
     └───────────────────────────────────────┘
                      │
        ┌─────────────┼─────────────┐
        ▼             ▼             ▼
 previousSibling   nextSibling   childNodes[n]


Explanation of Flow:

parentNode → Moves up one level in the DOM tree.
firstChild → Moves to the first child node of the current node.
lastChild → Moves to the last child node of the current node.
childNodes[n] → Accesses a specific child by index.
nextSibling → Moves to the next node at the same level.
previousSibling → Moves to the previous node at the same level.
